# python-web-cherrypy-api-postgres-simple

## Description
Simple web app that serves an api
for a bottle project.

Uses sqlalchemy query a table `dog`.

## Tech stack
- python
- cherrypy

## Docker stack
- python
- postgresql:alpine

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
